## 🧪 QUICK TEST (Copy-Paste)

### Create Booking:
```bash
curl -X POST http://localhost:8000/api/bookings \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "booking_type": "room",
    "booking_date": "2026-03-01",
    "start_time": "09:00",
    "end_time": "10:00",
    "purpose": "Team Meeting",
    "room_id": 1,
    "participants_count": 5
  }'
```

### Get Bookings:
```bash
curl http://localhost:8000/api/bookings \
  -H "Authorization: Bearer YOUR_TOKEN"
```

### Approve (Division):
```bash
curl -X PUT http://localhost:8000/api/bookings/1/approve-division \
  -H "Authorization: Bearer LEADER_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"action": "approve", "notes": "OK"}'
```

### Approve (DIVUM):
```bash
curl -X PUT http://localhost:8000/api/bookings/1/approve-divum \
  -H "Authorization: Bearer DIVUM_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"action": "approve", "notes": "OK"}'
```

## ⚠️ COMMON ERRORS & QUICK FIX

| Error | Solusi |
|-------|--------|
| Route not found | Run: `php artisan route:clear` |
| 401 Unauthorized | Token invalid/expired, login lagi |
| 403 Forbidden | Wrong role, check user.role di DB |
| 422 Validation | Check booking_date >= today, times valid |
| Foreign key error | room_id/vehicle_id must exist |
